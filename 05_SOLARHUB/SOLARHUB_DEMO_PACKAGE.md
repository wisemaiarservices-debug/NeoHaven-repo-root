# SolarHub Demo Package

## Demo Title

NOVA OS — AI Operating System for Sustainable Infrastructure

## One-Line Pitch

NOVA OS connects agriculture, energy, infrastructure, and AI decision support into one command center for renewable and agrivoltaic sites.

## Demo Objective

Show a complete SolarHub-ready scenario where NOVA observes an agrivoltaic site, analyzes risk, forecasts impact, recommends an action, simulates outcomes, and displays measurable benefits.

## GitHub-First Demo Stack

- NOVA Command Center: unified demo view
- NOVA Core AI v0: forecast, recommendation, scenario, explanation
- NeoAgro: agriculture and agrivoltaic intelligence
- NeoGrid: energy and battery optimization
- NeoCell: infrastructure readiness and edge telemetry

## Demo Scenario

A heatwave is forecast for an agrivoltaic site.

NOVA detects:

- Increased crop stress risk
- Declining soil moisture
- Higher irrigation demand
- Solar derating risk due heat
- Battery reserve needs
- Gateway and sensor readiness requirements

NOVA recommends:

- Run site readiness check
- Shift irrigation to evening window
- Charge battery during morning solar peak
- Reduce grid import during high-carbon periods
- Track expected water, carbon, and resilience impact

## Demo Flow

1. Open NOVA Command Center.
2. Show all services in fallback or connected mode.
3. Show NeoCell readiness and telemetry health.
4. Show NeoAgro crop and water risk.
5. Show NeoGrid solar, battery, and grid coordination.
6. Show NOVA Core AI v0 recommendation logic.
7. Show expected impact metrics.
8. Close with replication value for SolarHub pilots.
